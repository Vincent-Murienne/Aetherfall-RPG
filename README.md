# Aetherfall-RPG

Aetherfall est un jeu de rôle en console (RPG) développé en Python. Le
joueur choisit une classe, explore des zones, combat des ennemis, trouve
des trésors et tente de vaincre le Gardien du Donjon.

## Classes

-   Guerrier : fort et résistant
-   Mage : puissant avec la magie
-   Voleur : rapide et critique souvent

## Zones

-   Village : point de départ
-   Forêt : combats, trésors, clé du donjon
-   Donjon : boss final (verrouillé sans clé)

## Objectif

Trouver la clé dans la forêt, entrer dans le donjon et vaincre le
Gardien.

## Lancer le jeu

1.  Ouvrir un terminal dans le dossier du projet
2.  Lancer :

``` bash
python main.py
```

Pour jouer au jeux, tout ce fera dans le terminal. Il faudra utiliser votre clavier.
